В данном каталоге находятся файлы скомпилированной программы версии 2.0.0.

Для запуска:

Скачайте все файлы и запустите publish.part01.exe.

WinRAR выполнит распаковку всех архивов в одно место — после этого можно будет запустить программу.

После распаковки перейдите в папку publish и запустите файл OSTUVision.exe.

Разбиение на несколько архивов связано с особенностью GitHub (нельзя загрузить файл размером более 25 МБ).

Для работы программы требуется Visual C++ Redistributable, так как ONNX Runtime зависит от наличия Microsoft Visual C++ Redistributable.
