Программа OSTU Vision написана на C# (C‑Sharp) с использованием Python. Весь процесс разработки программы осуществлялся в VS Code.
Структура программы состоит из следующих разделов:
Каталог Forms. В данном каталоге находится файл MainForm.cs.
Каталог Helpers. В данном каталоге находится файл ImageUtils.cs.
Каталог Models. В данном каталоге находятся файлы:
CustomObject.cs;
DetectionResult.cs;
yolo26n.onnx;
yolo26n.pt.
Каталог Services. В данном каталоге находятся файлы:
IDetector.cs;
Logger.cs;
SpeechService.cs;
TrainingService.cs;
VideoProcessor.cs;
Yolo26Detector.cs.
Каталог Resources. В данном каталоге находятся:
logo.jpg;
railway.ico;
подкаталог traffic_lights. В этом каталоге находятся файлы с названием type.png (нумерация от 1 до 13), необходимые для отображения изображений светофоров в окне программы. Это поясняющая вкладка, описывающая 13 типов светофоров и их показания.
Файл appsettings.json.
Файл Program.cs.
Файл OSTUVision.csproj.
Файл train_yolo26.py.
Назначение программы
Программа предназначена для автоматического анализа фото и видео с целью распознавания железнодорожных светофоров, поездов и нестандартных объектов на железнодорожных путях.
Используемые технологии
Платформа: .NET 10.0.
Интерфейс: Windows Forms.
Нейросеть: YOLO26 (ONNX Runtime).
Обработка изображений: ImageSharp.
Синтез речи: System.Speech.
Формат данных: JSON.
Внешний вид программы и описание кнопок
Главное окно программы имеет 4 вкладки:
«Загрузить материал».
«Типы светофоров».
«Обучение модели».
«О программе».
Вкладка «Загрузить материал»
На этой вкладке есть возможность загрузить фото либо видео — для этого созданы соответствующие кнопки. После загрузки материала и нажатия кнопки «Анализировать» запускается процесс анализа. По завершении анализа пользователь может сохранить результат в любую папку.
По умолчанию обработанные фото и видео сохраняются по пути:
\OSTU\bin\Release\net10.0-windows\Results\Images (для фото)
\OSTU\bin\Release\net10.0-windows\Results\Video (для видео)
Поддерживаемые форматы данных:
Фото: .jpg, .jpeg, .png, .bmp, .webp, .gif, .tiff.
Видео: .mp4, .avi, .mkv, .mov, .wmv, .flv.
Вкладка «Обучение модели»
Эта вкладка позволяет:
добавлять типы классов;
обучать модель;
запускать тестовый режим (если обученная модель отсутствует).
Для обучения модели необходимо:
Ввести название класса.
Загрузить заранее подготовленный материал: фото с отрисованными объектами и текстовый файл с координатами.
Во время обучения в окне отображается прогресс выполнения.
Вкладка «О программе»
Вкладка содержит информацию о разработчиках и основных технологиях, используемых в программе.
